# symsquare
Draw symmetric square patterns

## Requirements

No third-party library required. Tested with Python 2.7 and Python 3.8.

## Usage

Run it with the size of the desired square canvas to be filled with
patterns. Example:

```shell
$ python symsquare.py 3
0
---
---
---
1
---
-X-
---
2
-X-
X-X
-X-
3
-X-
XXX
-X-
```
